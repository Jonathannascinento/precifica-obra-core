# Roadmap

## 0.1 — reliable core

- SINAPI CSV/XLSX normalization with diagnostics.
- Materials, labor charges, equipment, compositions, budgets, and BDI.
- Dual module packaging, tests, bilingual documentation, and secure release automation.

## 0.2 — format coverage

- Document additional real-world column mappings without redistributing source datasets.
- Add composition-specific parsing and richer validation summaries.
- Publish integration examples for APIs, scripts, and web applications.

## 0.3 — interoperability

- Stable adapters for persistence layers without coupling the core to a database.
- Import/export schemas and compatibility fixtures contributed by the community.
- Performance benchmarks for large monthly reference files.

Roadmap items are not promises. Public API changes follow semantic versioning.
